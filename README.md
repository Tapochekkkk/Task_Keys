# Doors

Keys and Doors

Демонстративный проект функции Interact в UE5.6 на Blueprints.

Управление:
WASD - передвижение
E - использовать

Использование:
На карте уже расставлены ключи, двери и постаменты.

Переменные BP_Postament:
Index - к какой двери привязан постамент

переменные BP_Door:
Index - индекс двери
NeedKeys - кол-во ключей для открытия двери

Требования:
Наличие UE5.6
FAQ

Зачем вы разработали этот проект?

Выполнение тестового задания

Задача: Открыть дверь при помощи двух ключей
Особые условия: Реализация должна быть только на BP, без использования плагинов.
